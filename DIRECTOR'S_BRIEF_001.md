# PRODUCTION WORKFLOW

## PRE-PRODUCTION (FREE FIRST)

Before using any paid AI video generation:

1. Create the complete storyboard.
2. Generate all master keyframe images using Nano Banana.
3. Use Nano Banana because image generation is free.
4. Generate one master image for each planned scene.
5. Review each image for realism, branding, lighting, composition, continuity, and character consistency.
6. If an image has issues, regenerate **ONLY** that image.
7. Do **NOT** continue until every image reaches premium Hollywood quality.
8. Present all approved keyframes for user approval.
9. Wait for explicit user approval before proceeding to any paid video generation.

## VIDEO GENERATION

After user approval:

- Use the approved Nano Banana keyframes as the visual reference for Higgsfield.
- Animate the approved images instead of creating scenes from scratch whenever supported.
- Generate one video shot at a time.
- Verify each completed shot before generating the next.
- Only regenerate a shot if there is a major issue such as broken anatomy, incorrect branding, inconsistent characters, severe AI artifacts, or unusable camera movement.
- Do not regenerate for minor artistic differences.

## CREDIT OPTIMIZATION

Always minimize paid AI generation.

- Use free tools whenever possible.
- Reuse approved fighters, arenas, lighting, camera settings, prompts, and environments.
- Lock character consistency.
- Lock environment consistency.
- Lock lighting consistency.
- Lock color grading consistency.
- Never create alternate versions unless requested.
- Avoid unnecessary generations.
- Save successful prompts as reusable templates.
- Prioritize the lowest Higgsfield credit usage while maintaining premium Hollywood-quality output.

## USER APPROVAL GATE

Before any paid Higgsfield generation:

**STOP.**

Present the Nano Banana images.

Wait for user approval.

Do not proceed until approval is received.

Cursor should manage the complete production pipeline automatically while using the fewest possible Higgsfield credits and maintaining premium Hollywood-quality output.
