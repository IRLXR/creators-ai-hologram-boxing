# 19 — QUALITY CONTROL

- [ ] Original imagery (not landing-page reuse for ads)
- [ ] No AI artifacts
- [ ] Correct arena (inflatable tent · approved silhouette)
- [ ] Correct headset (monogram strap · every spectator)
- [ ] Correct Wave (wireframe · faceless · WAVE waistband)
- [ ] Consistent lighting
- [ ] Consistent branding and color grade
- [ ] Characters remain consistent shot to shot
- [ ] Arena / Wave / headset proportions unchanged
- [ ] No flickering · no morphing
- [ ] No extra fingers, hands, or facial distortions
- [ ] Professional cinematic motion throughout
- [ ] Every Higgsfield shot matches its Nano Banana frame

## On failure

Regenerate **only** the affected frame or shot. Do not restart the full campaign.
