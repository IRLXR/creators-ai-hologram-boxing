# 20 — EXPORT GUIDE

## Deliverables

- Storyboards
- Master frames (`ads/master-frames/`)
- Videos (Seedance outputs → `ugc/output/` or campaign-specific folders)
- Posters
- Thumbnails
- Archive (manifest JSON + job IDs)

## Naming

| Asset | Pattern |
|-------|---------|
| Master frame | `NN-beat-name.png` |
| Ad plate | `[PLATFORM]-[campaign]-[beat].mp4` |
| Manifest | `MASTER-FRAMES-MANIFEST.json` |

## Archive fields

Record for each asset: job ID, model, prompt version, approval date, source URL.
