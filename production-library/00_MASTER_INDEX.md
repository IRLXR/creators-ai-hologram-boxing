## Core principles

- Never use landing-page images as production references.
- Generate all advertisement imagery from scratch.
- Preserve approved brand identity.
- Nano Banana generates master still frames.
- Higgsfield animates only approved master frames.
- Every advertisement must feel like a premium Hollywood technology launch.

## Production pipeline (mandatory)

1. Storyboard → Nano Banana master frames (free) → user approval
2. Seedance animation from approved frames only (paid)
3. QC per `19_QUALITY_CONTROL.md` → export per `20_EXPORT_GUIDE.md`
