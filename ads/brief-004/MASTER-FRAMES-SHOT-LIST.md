## How to use this list

1. Generate **one still per row** in order (Act 1 → Act 5).
2. QC every frame against the checklist below before marking `approved`.
3. Upload approved PNGs to Higgsfield → `media_confirm` → note `media_id` in manifest.
4. Only then run Seedance with `start_image` (separate prompt doc — next step).

**Headset rule**

- **Exterior (frames 01–06):** Headset **on** or **in hand / around neck** — never bare-headed adults in frame.
- **Interior (frames 07–13):** Headset **on every visible guest** — no exceptions.
- **Child in frame 13:** Headset optional (under 8); parent wears headset.

**Copy-paste block (interior frames)**

```
ABSOLUTE RULE — VR HEADSETS ON EVERY PERSON: Every visible human wears a sleek white VR headset in ski-goggle shape with three vertical pill-shaped sensors and a white luxury patterned head strap. Zero bare heads.
```

**Venue block (all frames)**

```
Black inflatable tent venue — NOT a dome, octagon, or boxing arena. Blue illuminated entrance on black shell. Photorealistic documentary night-out energy — authentic, never staged ad look.
```

## QC checklist (every frame)

- [ ] Inflatable **tent** — not dome / geodesic / MMA cage
- [ ] Headset rule satisfied for frame type
- [ ] No celebrity likenesses · no real fight recreations
- [ ] Emotion reads **authentic** — not stock-photo smile
- [ ] No text overlays burned into still (title card excepted)
- [ ] 16:9 composition · subject safe for 9:16 center crop
- [ ] Night lighting · urban · premium · not daylight carnival
