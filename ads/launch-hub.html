Step A — TikTok Business Center (one-time)

  1. Open business.tiktok.com → sign in with your TikTok account
  2. Create Business Center → name it Creators AI Hologram Boxing
  3. Link your TikTok profile @creatorsai (Assets → TikTok accounts → Add)
  4. Open ads.tiktok.com → Tools → Billing → add payment method
  5. Set bio link on TikTok to https://creators-ai-hologram-boxing.vercel.app/landing.html (Organic tab → bio field)

Step B — TikTok Pixel (recommended before spend)

  1. Events Manager → Web events → TikTok Pixel → Create pixel
  2. Setup method: Manually install pixel code (already on our landing page)
  3. Copy the Pixel ID (looks like CXXXXXXXXXXXXXXXXX)
  4. Paste into site-config.js → socialAds.tiktokPixelId → deploy to Vercel
  5. On the pixel, add event SubmitForm (fires when someone joins the waitlist)
  6. Test: open landing URL on phone → submit test email → Events Manager should show activity within ~20 min

No pixel yet? Use objective Traffic for Day 1, then switch to Website conversions once pixel is live.

Step C — Create your first campaign

  1. ads.tiktok.com → Create campaign
  2. Objective: Website conversions (pixel + SubmitForm) or Traffic (no pixel yet)
  3. Smart campaign: OFF (manual control for launch)
  4. Campaign name + budget from fields below · Ad group: US · 18–44 · English
  5. Placements: TikTok feed + TikTok search (automatic is fine)
  6. Upload the MP4 from the Start tab → paste ad text + URL below
  7. CTA button: Sign Up · Identity: your TikTok account
  8. Submit for review → turn campaign ON when approved (usually same day)
